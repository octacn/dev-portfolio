Classic Portfolio is a personal website built to represent a professional profile with a minimal and classic UI approach. The goal of the project was to showcase services, personal details, and contact information in a simple and visually pleasing format.

The website focuses on:

- Clean layout
- Easy navigation
- Clear typography
- Lightweight design

## Challenges Faced

- Making the design look professional with minimal UI elements.
- Managing layout consistency across different resolutions.
- Avoiding clutter while presenting multiple services clearly.
- Ensuring smooth user experience without heavy animations or frameworks.
